# Saudi GDP Trend Analyzer

This project analyzes Saudi Arabia's GDP from 2015-2025, calculates year-over-year growth, and generates visual charts. It demonstrates data analysis skills and AI-assisted insights using Claude AI.

**Author:** Muhammad Baqar, MA in Economics

---

## Project Overview

The Saudi GDP Trend Analyzer:

- Uses Python in Google Colab
- Loads and visualizes GDP data
- Calculates year-over-year growth
- Generates simulated AI-powered economic insights

This project was built through collaboration with Claude AI to explore Saudi economic trends and demonstrate practical applications of data analysis in economics.

---

## Installation & Usage

1. Open [Google Colab](https://colab.research.google.com)
2. Upload the provided `saudi_gdp.csv` file
3. Open the notebook `Saudi_GDP_Analyzer.ipynb`
4. Run all cells to reproduce analysis and charts
